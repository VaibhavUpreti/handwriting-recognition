# Handwriting Recognition

A handwriting recognition flask app.

# Requirements

1. Python version >= 3.8
2. CtCWordBeamSearch
3. SimpleHTR 

# Tech Stack

1. Flask
2. AWS EC2
3. Caddy webserver
